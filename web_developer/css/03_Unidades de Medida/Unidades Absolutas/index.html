Tipos Numéricos

Tipos de números utilizados classificados por medidas.

  • integer - 1024, -78;
  • number - 0.2, 128, -1.5;
  • percentage - 50%;
  • dimension -
    • length: distância (px, em, vw, in);
    • angle: ângulos(deg, rad, grad, turn);
    • time: tempo (s, ms);
    • resolutions: resolução (dpi, dpcm, dppx)

Medidas Absolutas

Medidas com valores únicos independentes dos elementos em que estão envolvidas. Não se alteram com base em elementos pai.

  • cm = centímetros
  • mm = milímetros
  • Q = quarto de milímetro
  • in = polegadas
  • pc = paica
  • pt = pontos
  • px = pixels
