Medidas Relativas

As unidades de relativas são calculadas com base em unidades já conhecidas de outros elementos, o que pode variar de acordo com a propriedade.

  • % - porcentagem. Baseia-se no elemento pai;
    • width: 300px;
    width: 50%;
  • em - Baseia-se na font-size do elemento (para outras propriedades) ou no elemento pai (para a propriedade font-size);
    • Elemento Pai
    font-size: 40px;
    Elemento Filho
    font-size: 2em ou 80px;
    width: 4em ou 320px;
  • rem - Tem o funcionamento similar ao em, contudo, utiliza o font-size de root (html) para definir os tamanhos das propriedades, por padrão o font-size do HTML é 16px;
    • Elemento Pai
    font-size: 40px;
    Elemento Filho
    font-size: 2rem ou 32px (HTML 16px);
    width: 18rem ou 288px;
  • vw - viewport width. Baseia-se na largura visualizada da página pelo usuáro no navegador;

    • viewport width: 50vw;

  • vh - viewport height. Baseia-se na altura visualizada da página pelo usuário no navegador;

    • viewport height: 10vh; viewport width: 50vw;

    Extras

    Unidades de medida pouco utilizadas.

  • ex - Baseia-se na altura do x;
  • ch - Baseia-se na largura do 0, em fontes monoespaçadas é definido a quantidade de caracteres por essa unidade;

    • Calculando medidas

    Utilizando a função calc(), obtemos a largura com a seguinte fórmula 100% = 300 - 40px -> 260px - 30px + 2em = 260px - 30px + 34px = 264px
