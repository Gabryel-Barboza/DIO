Seletores CSS

Bem-vindo!

No CSS nós possuimos os seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML
  • Seletor por ID (#): Busca elementos através do atributo ID
  • Seletor por Classe (.): Busca elementos pelo atributo classe
  • Seletor Universal (*): Seleciona todos os elementos do HTML
  • Seletor de Atributo ([atrib]) Seleciona elementos que possuem um atributo específico ou valor específico
