Compatibilidade entre Navegadores

É importante se atentar a esse ponto no desenvolvimento de sites. Existem alguns problemas de compatibilidade entre diferentes navegadores.

Motores de Renderização

Responsável pelo suporte a renderização de elementos nos navegadores. Determina a compatibilidade de renderização para estilos CSS. Exemplos:

  • Blink - Google Chrome, Opera
  • Gecko - Firefox
  • WebKit - iOS, Safari
  • Trident - Internet Explore

Veja se propriedades funcionam em determinado navegador no site caniuse

Prefixos no CSS

Permitem aplicar um estilo CSS experimental de navegadores, se diferenciando da propriedade padrão. Exemplos:

  • -moz-: Firefox
  • -webkit-: Chrome/Edge/Opera/Safari
  • -ms-: IE (Versão antiga)
  • -o-: Opera (Versão antiga)

São chamados de vendor prefixes, podem ser verificados em sites como caniuse para saber se determinadas propriedades ainda são experimentais e necessitam do prefixo.

  • -webkit-background-clip: text
