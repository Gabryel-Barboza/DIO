Padrões de Nomenclaturas

BEMCSS - Block, Element, Modifier
BEM faz alusão à POO, uma maneira de descrever a realidade no código, com uma variedade de padrões e uma maneira de pensar nas entidades do programa. -- Varga Stepanova
  • Bloco: Entidade independente (bloco de construção), Elemento pai, [Bloco] (.btn)
  • Elemento: Elemento filho, não independente, vinculado a um bloco, [Bloco__elemento] (.btn__price)
  • Modificador: modifica um bloco ou elemento, variante para alterar a aparência, propriedade, estado, [Bloco__elemento--modificador] (.menu__link--disabled)
SMACSS - Scalable and Modular Architect for CSS
Traz uma documentação de estruturação do CSS para auxiliar na organização em grandes projetos. --SMACSS
OOCSS - Object Oriented CSS
Obter os elementos usados e repetidos dentro da página como um objeto e usá-los quando necessário. Separa os estilos CSS em dois:
  • Estrutura: "width, height, padding, margin, position"
  • Design: "border, color, font, background"

Além de modularizar o código, deixando cada classe independente
