Pseudo-classe :indeterminate

Quando um elemento está no estado indeterminate, feito com JavaScript. O elemento progress fica no estado indeterminado, quando não possue valores padrões.

Pseudo-classe :invalid|valid|in-range|out-of-range

  • invalid: indica o estado de um campo inválido, ou incorreto, no formulário
  • valid: indica o estado válido de um campo
  • in-range: indica o estado de um campo com os valores dentro do limite definido
  • out-of-range: indica quando o campo está com valores fora do range definido
