Pseudo-elementos

Pseudo-elementos são utilizados para adicionar elementos na página sem a declaração no HTML, como também para selecionar algum componente do elemento

Identificado por ::palavra-chave

Pseudo-elemento - ::first-letter

Seleciona a primeira letra da linha.

Propriedades disponíveis:

  • Color
  • Background
  • Margin
  • Padding
  • Border
  • Font
  • Propriedades de texto

Pseudo-elemento - ::first-line

Seleciona a primeira linha do texto.

Propriedades disponíveis:

  • Color
  • Background
  • Font
  • Propriedades de texto

Pseudo-elemento - ::marker

Permite estilizar os tópicos de uma lista ou outros elementos

Propriedades aceitas
  • Color
  • Content
  • White-space
  • Font
  • Animation
  • Transition

Pseudo-elemento - ::placeholder

Selecione o placeholder de elementos de formulários

  • Color
  • Font
  • Background
  • Text

Pseudo-elemento - ::selection

Permite estilizar a seleção de texto

  • Color
  • Background-color
  • Text
