Serão apresentadas, separadas por tópicos, as marcações HTML aprendidas até o momento. Cada uma sendo apresentada com suas peculiaridades e atributos principais.

Tags Principais

São as marcações primordiais para o funcionamento de um documento HTML e permitem definir configurações importantes para o mesmo. Estão presentes em qualquer documento HTML.

  • <html> - Delimita todo o documento html e é definida a linguagem padrão do site com o atributo lang="". Possui fechamento </html>
  • <head> - Delimita o cabeçalho do documento, onde podem ser definidas configurações importantes para indicar o comportamento do site. Possui fechamento </head>
    • <title> - Indica o título que será apresentado na aba do site, na parte superior. Possui fechamento </title>
    • <meta> - São os metadados do documento, é utilizada para acrescentar e definir configurações, tais como o tamanho da tela utilizada pelo navegador e entre outros. Não possui fechamento.
  • <body> - Delimita o corpo do documento que será inserido todo o conteúdo, como esse que está sendo visualizado. Possui fechamento </body>

Tags de Conteúdo

São marcações utilizadas para acrescentar conteúdo a página, tais como textos, títulos, parágrafos, listas e mídias no geral.

  • <h1-h6> - Indica os títulos da página, possuindo níveis de 1 à 6 que organizam os títulos e subtítulos. Possui fechamento </h1-h6>
  • <p> - Delimita um novo parágrafo ao corpo do HTML, para cada parágrafo é utilizada uma nova linha. Possui fechamento </p>
  • <img> - Define um elemento de imagem no documento, com o local do arquivo descrito pelo atributo src="". Não possui fechamento.
  • <blockquote> - Delimita um espaço para uma citação, com o atributo cite="" pode-se referenciar o local da citação. Possui fechamento </blockquote>
  • <ul> - Define uma lista não ordenada, podem ser modificados os marcadores com o atributo type="". Possui fechamento </ul>
    • <li> - É o item da lista. Possui fechamento </li>
  • <ol> - Define uma lista ordenada, podem ser modificados os marcadores com atributo type="". Possui fechamento </ol>
    • <li> </li>
  • <a> - Vincula um outro elemento de hipertexto à marcação, em outras palavras, realiza uma ancoragem ao caminho definido por href="". Possui fechamento </a>

Tags de Estilo

São marcações que acrescentam estilos de fonte ao texto. Esses estilos sofreram uma alteração na parte semântica, criando-se novas tags para sua utilização.

  • <strong> - Adiciona o estilo de negrito ao texto, antes utilizada como <b> e </b>. Possui fechamento </strong>
  • <em> - Adiciona o estilo de itálico ao texto, antes utilizada como <i> e </i>. Possui fechamento </em>
  • <ins> - Adiciona o estilo de sublinhado ao texto, antes utilizado como <u> e </u>. Possui fechamento </ins>
  • <del> - Adiciona o estilo de removido ao texto, antes utilizado como <strike> e </strike>. Possui fechamento </del>
  • <mark> - Adiciona o estilo de texto marcado. Possui fechamento </mark>
  • <sup> - Adiciona o estilo de texto sobrescrito. Possui fechamento </sup>
  • <sub> - Adiciona o estilo de texto subscrito. Possui fechamento </sub>
  • <small> - Adiciona o estilo de texto pequeno. Possui fechamento </small>
  • <abbr> - Define uma caixa de texto para resolucionar a abreviação de algum elemento. Possui fechamento </abbr>
